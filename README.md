# hubot-who-am-i

Display username

See [`src/who-am-i.coffee`](src/who-am-i.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-who-am-i --save`

Then add **hubot-who-am-i** to your `external-scripts.json`:

```json
["hubot-who-am-i"]
```

## Sample Interaction

```
user1>> hubot who am i?
hubot>> user1
```
